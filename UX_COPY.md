## Voice & Tone Summary

### For Kids
- **BE:** Excited, encouraging, simple, fun
- **DON'T BE:** Condescending, preachy, boring, confusing
- **USE:** Emojis (sparingly), exclamation marks, "you/your"
- **AVOID:** Financial jargon, long sentences, passive voice

### For Parents
- **BE:** Helpful, clear, trustworthy, warm
- **DON'T BE:** Overly casual, corporate, pushy
- **USE:** Plain language, direct instructions
- **AVOID:** Talking down, overexplaining, guilt-tripping

### Emoji Guidelines
- âœ… Use for celebrations, milestones, positive feedback
- âœ… Use to soften error messages
- âœ… Use in headings/labels for visual interest
- âŒ Don't overuse (max 1-2 per message)
- âŒ Don't use in error codes or technical content
- âŒ Don't use randomly â€” each should add meaning
